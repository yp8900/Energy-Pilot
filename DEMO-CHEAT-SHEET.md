## 🔧 Troubleshooting During Demo

### If meters show offline:
- "This shows our comprehensive status monitoring"
- Check if demo data loaded: `npm run add-demo-meters`

### If page loads slowly:
- "Let me show you another feature while this loads"
- Use 24h time range instead of 30d

### If chart is empty:
- "Here's how the system looks with fresh installation"
- Navigate to a page with data

### If anomalies list is empty:
- "No anomalies detected - system is running smoothly!"
- Show trends or forecasts instead
